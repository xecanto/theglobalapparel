# Global Apparel – Premium Custom Apparel Manufacturing

Premium custom apparel manufacturing from Pakistan. Global Apparel provides full-stack production services including cut & sew, printing, and private labeling with low MOQs starting at just 50 pieces.

## 🚀 Live Demo
Visit the live site here: [https://xecanto.github.io/theglobalapparel/](https://xecanto.github.io/theglobalapparel/)

## ✨ Features
- **Visual Archive**: Interactive 8-row grid with multi-directional scroll parallax and mouse-based horizontal shifts.
- **Price Calculator**: Real-time production cost estimator with configurable garment specs, embellishments, and volume discounts.
- **Modern UI**: Dark-themed, premium aesthetic with smooth GSAP and Framer Motion animations.
- **Responsive Design**: Tailored experiences for Desktop, Tablet, and Mobile viewports.

## 🛠️ Technology Stack
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide Icons, Shadcn UI
- **Animations**: GSAP, Framer Motion
- **Deployment**: GitHub Pages

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/xecanto/theglobalapparel.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Build & Deploy
To build the production-ready application and deploy to GitHub Pages:
```bash
npm run build
npm run deploy
```
